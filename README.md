# snake
Snake game built with JavaFX and Maven.  

# Launch as executable
1. Download the snake.zip file inside the **executable** directory above and extract
3. Open `Terminal`.
4. Run the following commands to launch the application
   ```sh
   $ cd snake/bin
   $ ./launch.sh
   ```
