# snake
Snake game built with JavaFX and Maven.  

# Launch as executable (MacOS)
1. Download `snake.zip` from `exec` and extract.
3. Open `Terminal`.
4. Run the following commands to launch the application
   ```sh
   $ cd snake/bin
   $ ./launch.sh
   ```
